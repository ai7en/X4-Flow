# X4 Flow 📱⚡️

**X4 Flow** — это мобильное приложение-компаньон на базе Flutter, разработанное для Android-смартфонов. Его ключевое назначение — подготовка контента, кастомизация интерфейса и обслуживание автономных e-ink электронных книг (читалок) линейки **Xteink (модели x4 и x3)**, работающих на базе операционной системы экосистемы **CrossPoint**.

Приложение решает проблему изоляции e-ink устройств на базе микроконтроллеров (например, ESP32), предоставляя удобный мобильный интерфейс для тяжелых вычислительных задач (декодирование, ресайз графики, компиляция шрифтов).

---

## 🛠 Технологический стек и зависимости

* **Фреймворк:** Flutter SDK (Dart ^3.9.0), архитектура Material 3 с адаптивным темным/светлым оформлением.
* **Имя пакета в коде:** `myapp` (используется во внутренних импортах `package:myapp/...`).
* **Ключевые зависимости (`pubspec.yaml`):**
    * `image` & `archive` — низкоуровневая обработка растровой графики, алгоритмы сжатия и упаковка ZIP.
    * `xml` — парсинг метаданных электронных книг (FB2/OPF).
    * `webview_flutter` & `webview_flutter_android` — интеграция с веб-панелью читалки.
    * `dio` — работа с GitHub API для загрузки бинарных обновлений.
    * `file_picker` & `path_provider` & `permission_handler` — управление файловой системой на Android 10+ с обходом ограничений Scoped Storage.

## 🚀 Основные функциональные модули

### 📈 1. Конвертер и оптимизатор книг (`ConverterScreen`)
Инструмент для пакетной обработки литературы под слабые процессоры e-ink устройств. Работает в двух режимах:
* **FB2 ➔ ePUB (`fb2_to_epub.dart`):** Нативная конвертация книг. Содержит встроенный легковесный декодер кодировки `Windows-1251` для корректного чтения старых русскоязычных архивов. Автоматически генерирует правильные манифесты `content.opf` и таблицы оглавления (TOC).
* **EPUB Optimizer (`epub_optimizer.dart`):** Сканирует изображения внутри книги, переводит их в оттенки серого (Grayscale), удаляет альфа-канал прозрачности (смешивая пиксели с белым фоном) и физически уменьшает разрешение картинок под геометрию экранов X3/X4, предотвращая утечки RAM на читалке.

### 🖼 2. Генератор системных обоев (`WallpaperScreen`)
Создает монохромные изображения (BMP) строго под физические профили устройств (`device_profile.dart`): **X4** (480×800) и **X3** (528×792).
* **Режим Фото:** Загрузка картинок с кастомным кадрированием, зумом, изменением контраста/яркости и программным дизерингом (алгоритм Floyd-Steinberg) для отображения мягких e-ink градиентов.
* **Режим Цитат (`quote_templates.dart`):** Отрисовка цитат классиков (базы `quotes_ru.dart` и `quotes_en.dart`) на холсте с использованием 8 предустановленных стилей разметки (Классика, Инверсия, Рамка, Книжная страница, Минимализм и др.).
* **Режим Календаря (`calendar_templates.dart`):** Математическая отрисовка календарных сеток с возможностью инверсии цветов и функцией пакетной генерации архива обоев на все 12 месяцев года в один клик.

### 🔤 3. Компилятор шрифтов (`FontConverterScreen`)
Интерфейс для сборки файлов векторных шрифтов (TTF/OTF) в проприетарный бинарный растровый формат **`.cpfont`** для рендереров CrossPoint. Поддерживает одновременную упаковку 4 начертаний (Regular, Bold, Italic, BoldItalic), выбор диапазонов символов (ASCII, Кириллица, Латиница) и разрядности (1-Bit монохром или 2-Bit Grayscale).

### 🌐 4. Wi-Fi Передача данных (`TransferScreen`)
Модуль обмена данными с читалкой через локальный веб-интерфейс (`http://crosspoint.local`). Внутри Android WebView реализован глубокий перехват системного селектора файлов (`_setupAndroidFilePicker`), автоматически копирующий выбранные файлы в изолированный кэш приложения перед отправкой.

### 🔄 5. Менеджер прошивок (`FirmwareScreen`)
Парсер релизов GitHub API для экосистемы CrossPoint. Отслеживает репозитории `uxjulia/CrossInk`, `obijuankenobiii/inx`, `crosspoint-reader`, `franssjz/cpr-vcodex`, `alrudimgn` и других. Фильтрует и скачивает файлы `.bin` напрямую в публичную папку смартфона `/Download/XteinkFirmware` с использованием прав `MANAGE_EXTERNAL_STORAGE`.
