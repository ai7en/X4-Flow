# X4 Flow 🌊

Мобильное приложение-компаньон для E-Ink электронных книг **Xteink X4 / X3** на базе микроконтроллеров **ESP32** (работающих в экосистеме CrossPoint). 

Приложение разработано на Flutter и позволяет полностью подготовить контент (книги, обои, шрифты) и управлять прошивками устройства прямо со смартфона.

## 🚀 Основные возможности

Приложение разделено на 5 ключевых модулей (вкладок):

1. **📚 Конвертер книг (FB2 ➔ ePUB)**
   * Локальная конвертация файлов `.fb2` в оптимизированный формат `.epub`.
   * Интегрированный **EpubOptimizer**: автоматическое пережатие изображений в оттенки серого (Grayscale) и ресайз под точное разрешение экрана вашего устройства для экономии памяти ESP32.
   * Корректная обработка структуры книги и сносок (footnotes).

2. **🎨 Генератор E-Ink Обоев**
   * **Фото**: Загрузка пользовательских картинок с возможностью кадрирования, поворота, настройки яркости/контрастности и наложения дизеринга (Dithering) для качественного отображения на экранах с ограниченной палитрой.
   * **Цитаты**: Генерация стильных обоев со встроенными литературными цитатами (на русском и английском языках) с выбором из 8 уникальных шаблонов оформления.
   * **Календарь**: Создание обоев-календарей на текущий месяц с выделением выходных дней.

3. **🔤 Конвертер Шрифтов (.cpfont)**
   * Локальная сборка и компиляция десктопных шрифтов в бинарный формат `.cpfont`, используемый читалками CrossPoint.
   * Одновременная упаковка **4 стилей** (Regular, Bold, Italic, BoldItalic).
   * Поддержка **2-Bit сжатия** (4 оттенка серого) для сглаживания шрифтов на E-Ink дисплее.
   * Управление диапазонами символов (ASCII, Кириллица, Латиница) для минимизации итогового размера файла.

4. **🔄 Менеджер прошивок (Релизы)**
   * Парсинг актуальных OTA-сборок (`.bin`) напрямую из официальных GitHub-репозиториев экосистемы:
     * `CrossInk (uxjulia)`
     * `Inx (obijuankenobiii)`
     * `CrossPoint Reader`
     * `vCodex (franssjz / alrudimgn)`
     * `Papyrix`
     * `AALU`
   * Скачивание файлов прошивок во внутреннюю память телефона для последующей загрузки.

5. **🌐 Локальная Wi-Fi Панель**
   * Встроенный WebView для быстрого подключения к веб-интерфейсу читалки (`http://crosspoint.local` или по IP-адресу).
   * Позволяет загружать книги на устройство без проводов по воздуху.

## 📱 Поддерживаемые устройства

| Модель | Разрешение | Профиль в коде |
| :--- | :--- | :--- |
| **Xteink X4** | 480 × 800 | `DeviceModel.x4` |
| **Xteink X3** | 528 × 792 | `DeviceModel.x3` |

## 🛠 Технологический стек

* **Framework:** Flutter (Канал Stable)
* **Язык:** Dart (с поддержкой Null Safety)
* **Архитектура:** Стейтфул-компоненты с кастомной локализацией (`AppLocalizations` RU/EN).
* **Ключевые зависимости:**
  * `image` — обработка изображений, дизеринг и Grayscale.
  * `archive` — упаковка/распаковка EPUB (ZIP) архивов.
  * `xml` — парсинг структуры FB2, OPF-манифестов и контейнеров.
  * `dio` — скачивание прошивок с GitHub API.
  * `webview_flutter` — интеграция веб-панели управления.
  * `shared_preferences` — сохранение настроек языка, темы и GitHub-токенов.
